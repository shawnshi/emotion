# Emotion Tracking App

A responsive web application for tracking emotions, symptoms, and energy levels. Built with React, TypeScript, and Tailwind CSS.

## Features

- **User Signup**: A clean, dark-themed signup page.
- **Overview Dashboard**: Visualizes mood and energy trends over time using charts.
- **Daily Log**: Allows users to log daily symptoms, mood, energy levels, and journal notes.
- **Responsive Design**: Optimized for mobile and desktop viewing.

## Tech Stack

- **Framework**: [React](https://react.dev/) with [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Charts**: [Recharts](https://recharts.org/)
- **Routing**: [React Router](https://reactrouter.com/)

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm

### Installation

1. Clone the repository (if applicable) or download the source code.
2. Install dependencies:

   ```bash
   npm install
   ```

### Running Development Server

To start the local development server:

```bash
npm run dev
```

The application will be available at `http://localhost:5173` (or the port shown in your terminal).

### Building for Production

To create a production build:

```bash
npm run build
```

The built artifacts will be in the `dist/` directory.

### Preview Production Build

To preview the production build locally:

```bash
npm run preview
```
